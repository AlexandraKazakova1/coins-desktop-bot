## Інструкція користувача

CoinsBot — напівавтоматичний бот покупки

1. Що це за програма

CoinsBot — це ПК-програма для Windows, яка допомагає швидко натиснути кнопку “Купити” на сайті
👉 https://coins.bank.gov.ua/

Програма не обходить захисти і не вводить логіни/паролі — авторизація та перевірки проходяться вручну.

2. Що потрібно перед початком

Windows 10 / 11

Встановлений Google Chrome

Інтернет-зʼєднання

Обліковий запис на сайті coins.bank.gov.ua

3. Запуск програми

Запусти файл CoinsBot.exe

Відкриється невелике вікно програми

4. Кнопки та поля
🔹 Авторизація (крок 1)

Натисни кнопку «Авторизація»

Відкриється браузер Chrome

Перед цим задай список акаунтів і кількість вкладок на кожен акаунт.

Увійди в обліковий запис вручну (для кожного запущеного профілю)

Якщо ти вже авторизована — нічого робити не потрібно

❗ Важливо: браузер не закривати після авторизації

🔹 Додати вкладку на цей акаунт (крок 2)

Після авторизації натисни кнопку «Додати вкладку на цей акаунт».

Для кожної вкладки зʼявиться окрема картка зі своїм:
- полем URL товару
- кнопкою запуску пошуку кнопки «Купити»
- кнопкою зупинки
- окремим статусом

Ти можеш додати до 10 вкладок і запускати їх по одній або всі разом.

🔹 Дата і час старту (необовʼязково)

Якщо вказати дату і час:

бот відкриє сторінку заздалегідь

і клікне “Купити” точно у вказаний момент

Якщо залишити поле порожнім:

бот працює в режимі Standby

чекає, поки кнопка “Купити” зʼявиться / стане активною

і клікає одразу

🔹 Почати відстеження

Запускає роботу бота (або окремої вкладки, або всіх вкладок одразу)

У нижній частині зʼявляється статус

🔹 Зупинити

Повністю зупиняє роботу бота

Закриває браузер

5. Статуси програми
   Статус Що означає
   Готово Програма нічого не робить
   Авторизація Потрібно увійти на сайт
   Standby Очікує появу кнопки “Купити”
   Озброєно Очікує заданий час старту
   Підготовка Відкриває сторінку товару
   Старт! Натискає “Купити”
   Товар додано в кошик Успіх ✅
   Потрібна перевірка Пройди перевірку вручну
   Помилка Перевір повідомлення
6. Якщо зʼявилась перевірка “Я не робот”

Це нормально

Пройди перевірку вручну в браузері

Програма автоматично продовжить роботу

❗ Не оновлюй сторінку і не відкривай товар в іншій вкладці

7. Корисні поради

Не використовуй VPN або проксі

Не закривай Chrome під час роботи бота

Краще запускати програму заздалегідь

Якщо щось пішло не так — натисни «Зупинити» і запусти знову

8. Безпека

Програма не зберігає паролі

Не має доступу до платіжних даних

Працює лише в межах відкритого браузера

9. Обмеження

Успіх залежить від:

навантаження сайту

обмежень покупки

рішень самого сайту

Програма не гарантує 100% результат, але значно підвищує шанс встигнути
🔹 Акаунти

Можна ввести декілька акаунтів через кому або новий рядок.

Для кожного акаунта програма створює окремий Chrome-профіль.

🔹 Кількість вкладок на акаунт

Можна вказати скільки паралельних вкладок запускати для кожного акаунта (1–10).
